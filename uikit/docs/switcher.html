切换器/Switcher

通过不同的内容窗格进行动态变换。

用法

切换器由数个触发单元和与他们相关联的内容项组成。添加 data-uk-switcher="{connect:'#ID'}" 属性到包含触发单元的元素，并且指向包含内容项的元素的id。添加 .uk-switcher 类到同一个元素。通常，切换器和其他组件搭配使用，这里将展示其中一部分。

Code.

<!-- 这是触发单元的容器 -->
<ul data-uk-switcher="{connect:'#my-id'}">
    <li><a href="">...</a></li>
</ul>

<!-- 这是内容项的容器 -->
<ul id="my-id" class="uk-switcher">
    <li>...</li>
</ul>

注意 效果组件 中的 .uk-margin 类被用在一些例子中，用于在切换器中添加适当的间距。

切换器与子导航

切换器简单地应用于 子导航组件 中。

</ul>

<!-- 这是内容项的容器 -->
<ul id="my-id" class="uk-switcher">
    <li>...</li>
</ul>

切换器与选项卡

作为一个特例，添加使用"tab"参数替代 "switcher"的 data-uk-tab="{connect:'#ID'}" 属性到选项卡式的导航中，使得切换器和 选项卡组件 相互结合。

示例

Code.

<!-- 这是包含触发单元的选项卡式导航T -->
<ul class="uk-tab" data-uk-tab="{connect:'#my-id'}">
    <li><a href="">...</a></li>
</ul>

<!-- 这是内容项的容器 -->
<ul id="my-id" class="uk-switcher uk-margin">
    <li>...</li>
</ul>

注意 当使用选项卡组件的 .uk-tab-bottom 类的时候，导航和内容需要按相反的顺序放置，这样内容才会显示在选项卡的上方。

垂直选项卡

使用 网格组件正确地显示内容以及带有垂直排列的选项卡导航。

例子

  • Hello again!
  • Bazinga!
  • Tab
  • Tab
  • Tab

  • Bazinga!

Code

<!-- 这是左侧垂直的选项卡式导航 -->
<div class="uk-grid">
    <div class="uk-width-medium-1-2">
        <!-- 这是包含了触发单元的垂直选项卡式导航 -->
        <ul class="uk-tab uk-tab-left" data-uk-tab="{connect:'#my-id'}">...</ul>
    </div>
    <div class="uk-width-medium-1-2">
        <!-- 这是内容项的容器 -->
        <ul id="my-id" class="uk-switcher">...</ul>
    </div>
</div>

<!-- 这是右侧垂直的选项卡式导航  -->
<div class="uk-grid">
    <div class="uk-width-medium-1-2 uk-push-1-2">
        <!-- This is the vertical tabbed navigation containing the toggling elements -->
        <ul class="uk-tab uk-tab-right" data-uk-tab="{connect:'#my-id'}">...</ul>
    </div>
    <div class="uk-width-medium-1-2 uk-pull-1-2">
        <!-- This is the container of the content items -->
        <ul id="my-id" class="uk-switcher">...</ul>
    </div>
</div>

切换器与按钮

切换器同样可以应用于 按钮或按钮组 中。只需要添加切换器data属性到一个包含了一组按钮的 <div> 元素中或者添加到带有 .button-group 类的元素。

示例

Link
  • Hello!
  • Hello again!
  • Bazinga!
Link
  • Hello!
  • Hello again!
  • Bazinga!

Code.

<!-- 这是用于切换的按钮的容器 -->
<div data-uk-switcher="{connect:'#my-id'}">
    <button class="uk-button" type="button">...</button>
</div>

<!-- 这是包含了内容项的容器 -->
<ul id="my-id" class="uk-switcher uk-margin">...</ul>

<!-- 这是包含了用于切换的按钮的按钮组 buttons -->
<div class="uk-button-group" data-uk-switcher="{connect:'#my-id'}">
    <button class="uk-button" type="button">...</button>
</div>

<!-- 这是包含了内容项的容器  -->
<ul id="my-id" class="uk-switcher uk-margin">...</ul>

切换器与导航菜单

若需将切换器应用于 导航菜单组件，添加data属性到导航菜单的 <ul> 元素中。使用 网格组件 将导航菜单和内容项放置在同一个网格布局中。

示例

  • Item

  • Item
  • Hello!
  • Hello again!
  • Bazinga!

Code.

<div class="uk-grid">
    <div class="uk-width-medium-1-4">

        <!-- 这是包含了触发单元的导航菜单 -->
        <ul class="uk-nav uk-nav-side" data-uk-switcher="{connect:'#my-id'}">
            <li><a href="">...</a></li>
        </ul>
    </div>
    <div class="uk-width-medium-3-4">

        <!-- 这是内容项的容器 -->
        <ul id="my-id" class="uk-switcher">
            <li>...</li>
        </ul>
    </div>
</div>

JavaScript 事件

每当切换器组件切换到一个内容区域时，都会触发一次 uk.switcher.show 事件。

示例

$('[data-uk-switcher]').on('uk.switcher.show', function(event, area){
    console.log("Switcher switched to ", area);
});
