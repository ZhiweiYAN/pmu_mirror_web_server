创建一个主题

扩展UIkit并添加你自己的独特风格样式。

如何开始

默认情况下，UIkit带有一个非常基础的风格。如果要修改这个风格或添加自己的风格样式，你不需要进行编辑任何的核心框架文件。相反，你可以通过创建一个自定义主题来扩展UIKit。这使你可以轻松地更改UIKit，因为这些修改独立于核心部分。我们甚至提供了一些灵巧的主题以帮助你入门。要创建自己的主题，请按照下面步骤操作：

1. 创建主题目录

下载或克隆UIKit，安装 Node 和 Grunt，最后在这里创建你的主题文件夹 /themes/custom/THEME-NAME。

注意 对 /custom 文件夹设置为 git ignore，这样可以防止你的自定义文件被推入到UIKit库中。

2. 添加你的主题

创建 uikit.less 文件并添加到 /themes/custom/THEME-NAME 文件夹下 @import "../../../src/less/uikit.less"; 从默认的框架删除所有LESS文件，采用它的基本样式。就是这样！你可以开始添加你自己的样式到你的新主题中了。

注意 为了让你马上开始，我们提供了默认、渐变和扁平化风格的三个主题样式。默认情况下所有的文件都已经导入了，你也可以找到所有的主题文件及与它们相关的钩子。所以你需要做的是复制和重命名文件夹到 /custom 目录。

3. 定制和测试

在 UIKit 文件夹中运行 grunt 的 indexthemes 任务。现在，新创建的主题将自动出现在定制器和测试文件中。

最佳主题实践

设计你的主题有多种不同的方法，我们推荐以下的工作流程。

1. 使用变量

首先要做的是自定义已声明的变量的值。你可以从核心框架里面找到每个组件的 LESS 变量文件 并重写覆盖它们到你的主题。

示例

/src/less/button.less

// 默认值
@button-height: 30px;

/themes/custom/THEME-NAME/button.less

// 新的值
@button-height: 35px;

2. 使用钩子

为了防止架空选择器，我们使用 LESS 的混合（Mixins）方法与 UIkit 源码中预定义的选择器进行挂钩，并运用其附加属性。这样选择器即可很容易的贯穿所有文档和全局的变换修改。

示例

/src/less/panel.less

// CSS规则
.uk-panel {
    background: @background;

    // 混合增加新的声明
    .hook-panel;
}

/themes/custom/THEME-NAME/panel.less

// 混合增加新的声明
.hook-panel() { color: #fff; }

3. 其他钩子

如果没有变量也没有钩子可用，当然你也可以自行创建一个你自己的选择器。如果要执行这项，请使用 .hook-button-misc 钩子在里面写入你的规则样式，这将编译你新选择的CSS文件用于合适的地方。

示例

/themes/custom/THEME-NAME/panel.less

// 其他混合
.hook-button-misc() {

    // 新规则
    .uk-panel a { color: #f00; }
}
