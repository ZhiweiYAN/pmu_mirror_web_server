选项卡

创建一个不同风格样式的选项卡式导航。

用法

该选项卡组件由并排排列的可点击的标签组成。

类 概述
.uk-tab 添加这个类到 <ul> 元素中，定义选项卡组件。使用 <a> 元素作为列表中的标签项。
.uk-active 为列表项添加此类，对其应用选中状态。
.uk-disabled 为列表项添加此类，对其应用禁用状态。.

data-uk-tab 属性是为了实现两个目的而必须添加的。首先，它可以使其应用相应行为。在较窄的视口中，比如手机，之前的标签将被合并成一个下拉菜单，由一个单独的标签来进行标签项切换。这将需要用到 下拉菜单组件 才能正常运行。

其次，它的功能将耦合到 切换器组件。这是使用选项卡式导航动态切换不同内容所必须的。

示例

  • Active
  • Item
  • Item
  • Disabled

Code.

<ul class="uk-tab" data-uk-tab>
    <li class="uk-active"><a href="">...</a></li>
    <li><a href="">...</a></li>
    <li><a href="">...</a></li>
    <li class="uk-disabled"><a href="">...</a></li>
</ul>

水平修饰类

添加以下类之一来改变选项卡的样式。这些修饰类也可以组合使用。

对齐选项卡

类 概述
.uk-tab-flip 添加这个类将选项卡右侧对齐并颠倒顺序。
.uk-tab-bottom 添加这个类到 <ul> 元素将选项卡放置底部。这也可以与其它修饰类结合使用。

示例

  • Active
  • Item
  • Item
  • Active
  • Item
  • Item

Code.

<ul class="uk-tab uk-tab-flip" data-uk-tab>...</ul>
<ul class="uk-tab uk-tab-bottom" data-uk-tab>...</ul>

两端对齐选项卡

添加 .uk-tab-grid 类和 网格组件 中的一个 .uk-width-* 类到列表项中，将选项卡放置一个网格中，使其占满父元素的宽度。

示例

  • Active
  • Item
  • Item

Code.

<ul class="uk-tab uk-tab-grid uk-tab-bottom" data-uk-tab>
    <li class="uk-width-1-3"><a href="">...</a></li>
</ul>

居中选项卡

添加 .uk-tab-center 类到一个包含选项卡导航的 <div> 元素中，使选项卡居中。

示例

  • Active
  • Item
  • Item

Code.

<div class="uk-tab-center">
    <ul class="uk-tab" data-uk-tab>...</ul>
</div>

垂直修饰类

添加 .uk-tab-left 或 .uk-tab-right 类 垂直对齐到选项卡的左侧或右侧。

示例

  • Active
  • Item
  • Item
  • Active
  • Item
  • Item

Code.

<ul class="uk-tab uk-tab-left uk-width-medium-1-2">
    <li>...</li>
</ul>
<ul class="uk-tab uk-tab-right uk-width-medium-1-2">
    <li>...</li>
</ul>

Responsive behavior

In narrower viewports, like phones, vertical tabs align themselves horizontally. Adding the data-uk-tab attribute will apply the same responsive behavior as horizontal tabs.

Tabs with dropdowns

This is an example of how to use tabs with the Dropdown component.

示例

  • Active
  • Item
  • Item
  • More
    • Item
    • Another item
    • Header
    • Item
    • Another item
    • Separated item

Code.

<ul class="uk-tab">
    <li><a href="">...</a></li>

    <!-- This is the container enabling the JavaScript -->
    <li data-uk-dropdown="{mode:'click'}">

        <!-- This is the menu item toggling the dropdown -->
        <a href="">...</a>

        <!-- This is the dropdown -->
        <div class="uk-dropdown uk-dropdown-small">
            <ul class="uk-nav uk-nav-dropdown">
                <li><a href="">...</a></li>
            </ul>
        </div>
    </li>
</ul>
