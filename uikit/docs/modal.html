模态对话框

创建不同样式和过渡形式的模态对话框。

用法

模态对话框由一个遮罩层，一个对话框和一个关闭按钮组成。

类 概述
.uk-modal 添加这个类到一个 <div> 元素中，创建一个对话框容器和屏蔽页面的遮罩层。添加一个id来标示这个元素对于触发遮罩层至关重要。
.uk-modal-dialog 添加这个类到一个子级的 <div> 元素中创建对话框。
.uk-modal-close 添加这个类到一个 <a> 或 <button> 元素中创建一个对话框中的关闭按钮。我们推荐添加 关闭组件 中的 .uk-close 类给关闭按钮设置一个恰当的样式，你也可以使用文字或者图片。

你可以使用任意元素来触发模态对话框。一个 <a> 元素需要链接到模态对话框的id。为了使必要的JavaScript能够生效，需要添加 data-uk-modal 属性。如果你使用的是其他的元素，比如按钮，只需添加 data-uk-modal="{target:'#ID'}" 属性来瞄准模态对话框的id。

Code.

<!-- 这是一个用来切触发模态对话框的锚 -->
<a href="#my-id" data-uk-modal>...</a>

<!-- 这是用来触发模态对话框的按钮 -->
<button class="uk-button" data-uk-modal="{target:'#my-id'}">...</button>

<!-- 这是模态对话框 -->
<div id="my-id" class="uk-modal">
    <div class="uk-modal-dialog">
        <a class="uk-modal-close uk-close"></a>
        ...
    </div>
</div>

JavaScript 选项

默认情况下，点击模态遮罩层时，模态对话框会自动关闭。要想避免它的发生，添加 data-uk-modal="{target:'#ID',bgclose:false}" 属性即可。

无边框调整

默认情况下，模态对话框有一个padding填充。为了避免这种情况和消除所有的框，添加一个 .uk-modal-dialog-frameless 类。如果你想用模态对话框来展示图片的话，这会很有用。关闭按钮会自动调整添加到无边框对话框的合适位置上。

示例

Code.

<!-- 这是触发模态对话框的锚 -->
<a href="#my-id" data-uk-modal>...</a>

<!-- 这是模态对话框 -->
<div id="my-id" class="uk-modal">
    <div class="uk-modal-dialog uk-modal-dialog-frameless">
        <a href="" class="uk-modal-close uk-close uk-close-alt"></a>
        <img src="" alt="">
    </div>
</div>

注意 在无边框的模态对话框内创建关闭按钮时，我们同样建议添加 关闭组件 中的 .uk-close 类。在这种情况下，你还必须要添加 .uk-close-alt 类给按钮添加适配无边框模态对话框的样式。

把网页容器的宽度应用到模态对话框上，添加 .uk-modal-dialog-large 类即可。

在模态对话框中溢出容器

JavaScript

你可以通过原生的Javascript处理模态对话框。

示例

var modal = $.UIkit.modal(".modalSelector");

if ( modal.isActive() ) {
    modal.hide();
} else {
    modal.show();
}

事件

当模态对话框处于打开状态时，触发一次 uk.modal.show 事件。当模态对话框关闭状态时，触发 uk.modal.hide 事件。

示例

$('.modalSelector').on({

    'uk.modal.show': function(){
        console.log("Modal is visible.");
    },

    'uk.modal.hide': function(){
        console.log("Element is not visible.");
    }
});
