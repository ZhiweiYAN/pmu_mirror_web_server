附加组件

所有附加组件都是独立的，并不包含在 UIKit 的核心框架中。

UIKit 提供了一些高级的组件，它们不包含在 UIKit 的核心框架中。通常你在普通的网站上使用不到这些插件。如果你创建高级的用户界面管理等领域，它们会派上用场，比如“拖放排序”和“日期选择器”。

用法

当你 下载 UIkit 后，你可以发现附加组件在 /addons 文件夹中。下面是附加组件 “日期选择器” 的文件结构。

/addons/css
    <!-- 这个日期选择器附加组件为基本风格的样式 -->
    datepicker.css
    datepicker.min.css
    ...

    <!-- 这个日期选择器附加组件具有渐变风格的样式 -->
    datepicker.gradient.css
    datepicker.gradient.min.css
    ...

    <!-- 这个日期选择器附加组件具有扁平化风格的样式 -->
    datepicker.almost-flat.css
    datepicker.almost-flat.min.css
    ...

/addons/js
    <!-- JavaScript和压缩后的版本 -->
    datepicker.js
    datepicker.min.js
    ...

若要使用其中的附加组件，你只需要把它的JavaScript和CSS文件添加到你的文档头部。

示例

<!DOCTYPE html>
<html>
    <head>
        <title></title>
        <link rel="stylesheet" href="uikit.min.css" />
        <script src="jquery.js"></script>
        <script src="uikit.min.js"></script>
        <!-- Datepicker add-on -->
        <script src="datepicker.js"></script>
        <link rel="stylesheet" href="datepicker.css">
    </head>
    <body>
    </body>
</html>
