创建一个样式

扩展UIkit并添加你自己的独特风格样式。

仅仅通过一组变量就可以改变现有主题的外观。你可以为每个主题创造无限的风格。要创建你自己的风格，请按照下列步骤操作：

1. 创建样式目录

在这里创建你的风格文件夹 /themes/custom/THEME-NAME/styles/STYLE-NAME.

2. 添加你的样式

创建一个空白 style.less 文件到 /themes/custom/THEME-NAME/styles/STYLE-NAME 文件夹。现在你可以手动添加新的变量声明或者使用 定制器。当您完成时，点击 下载 LESS 按钮保存在你的样式文件到文件夹中。

3. 定制和测试

在UIKit的目录运行 grunt task 的 indexthemes 就可以进行定制或测试你新创建的样式。

4. 持续定制

现在，你可以继续 自定义 你的样式。
